# 02 · 功能说明（用户能做什么）

## 奥扫

- 按**场景 + 子类**筛选（像电影类型标签，不用手打关键词为主）  
- **距离滑条**（默认 15km）、排序：比便宜 / 性价比 / 质价比  
- **民宿**：先「开始搜索」浏览店铺，**不必先选日期**；可选再查档期  
- 点卡片 → **店家详情**：实景图、环境标签、价格、店家介绍、外链比价、底部可选地图导航  
- 点大图/按钮 → **新窗口**打开官网/携程/点评（关掉标签回到应用）  
- 卡片图角 **♥ 收藏**、**⚖ 对比**（最多 3 家，**必须同类**：同 scene + 同子类）  
- 对比维度：**环境、价格、距离**（直线，非驾车里程）

## 暇兜兜

暇兜兜机制（冷却恢复制）：
- 每次打开海面，随机生成 8-12 只瓶子
- 捞1只扣1只，捞完为止
- 每3分钟自动恢复1只（未达15只上限时持续恢复）
- 最多攒到15只
- 手动刷新按钮：30分钟冷却，点后补满至12只
- 空瓶概率10%（幽默文案）
- 每日第一只捞到的瓶子，必不是空瓶，签文带"今日签文"标识

- 选瓶子类型（场景+子类）+ 距离  
- 捞到后：**「走进这家店看看」** → 详情（签文、介绍、价格、服务），**不自动跳导航**  
- **扔回海里**：动画 + 短音效，再捞下一只  
- 详情里大图**不再一点就外链**，外链在下方「去平台看更多」

## 我的

- 头像、昵称  
- **收藏**列表，点卡片进详情  
- **工厂/二手一日游**：收藏 ≥2 家带坐标且标签匹配时，可 **串联路线 1→2→3** + 高德多途经  
- **捞瓶历史**  
- **许愿**（想找但还没有的店）  
- **上传嘎算优惠**：**家人发现的好价**，谁都能传，**不是**商家承包入驻；最后一栏是**提供情报的昵称**  
- 提交：优先后端 `POST /api/ugc`；失败则 **存本机** localStorage，奥扫/暇兜兜仍能搜到

## 地图 / 导航

- 详情 →「地图/导航（可选）」→ 全屏地图页，顶/底 **← 返回**，手机返回键支持  
- 可再开高德驾车（新窗口）  
- 直线距离 + 方位示意，**不是**应用内转弯导航
