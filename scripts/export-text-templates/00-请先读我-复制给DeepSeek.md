# 介嘎算（ga-ge-soen）— 给 DeepSeek 的文字资料包

> 本文件夹**全是可复制文字**（Markdown、JS/JSX 源码文本、JSON 数据），**没有**微信小程序、没有编译后的网页二进制包。  
> 版次：**0.3.0** · 生成后请从子文件夹按需复制段落给 DeepSeek。

---

## 建议你怎么喂给 DeepSeek

1. 先复制 **`文档/01-项目总览.md`** + **`文档/02-功能说明-用户能做什么.md`**
2. 若要改功能，再复制对应 **`源码-前端/...`** 或 **`源码-后端/...`** 里的单个文件全文
3. 数据规则看 **`文档/05-数据字段说明.md`**，接口看 **`文档/04-API接口.md`**

---

## 文件夹结构

| 文件夹 | 内容 |
|--------|------|
| **文档/** | 中文说明（产品、功能、API、近期改动）— **优先复制这些** |
| **源码-前端/** | React H5 全部 `.jsx` `.js` `.css` 文本 |
| **源码-后端/** | Express API、种子数据 JSON、路由 |
| **项目根说明/** | VERSION、README、PATHS 等 |

## 产品一句话

**介嘎算** = 上海奉贤区家庭生活优惠发现工具（家庭自用，非对外商业小程序）。三个 Tab：**奥扫**（主动搜）、**暇兜兜**（漂流瓶捞优惠）、**我的**（收藏、上传、许愿）。
