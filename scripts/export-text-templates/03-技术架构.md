# 03 · 技术架构

## 前端入口

- `main.jsx` → `App.jsx` → `AppProvider`（`context/AppContext.jsx`）  
- `BottomNav.jsx` 切换：`aoso` | `xiadoudou` | `profile`  
- 全局状态：定位、偏好、搜索结果、收藏、对比、漂流瓶、许愿、UGC API

## 关键模块

| 模块 | 文件 | 作用 |
|------|------|------|
| 搜索 | `AoSoPage.jsx` + `localDeals.js` | 本地种子查询；有后端时 `searchDeals` |
| 详情 | `DealDetailSheet.jsx` | 店家介绍、外链、导航 |
| 比价 | `CompareModal.jsx` + `compareRules.js` | 同类校验、三维表 |
| 路线 | `NavRouteModal.jsx` + `useHistoryBackClose.js` | 返回栈 |
| 一日游 | `FavoriteTripModal.jsx` + `visitRoute.js` | 收藏串联 |
| 外链 | `dealLinks.js` | 官网/平台 URL 兜底 |
| 暇兜兜 | `XiaDouDouPage.jsx` + `bottleLimits.js` | 无限捞、本机历史 |
| UGC | `localUgc.js` + `ProfilePage` UgcModal | 本机兜底 |

## 后端入口

- `server.js`：CORS、JSON、`/api/deals`、`/api/bottle`、`/api/ugc`、`/api/wish`、异步 `/api/search`  
- `dealsStore.js`：种子 JSON + UGC 合并查询  
- `routes/bottle.js`：随机捞瓶 + 签文  
- `routes/ugc.js`：家人上传

## 数据流（无后端时）

```
dealsSeed.json + localStorage(UGC) 
  → queryLocalDeals() 
  → 奥扫列表 / 暇兜兜随机
```

## 数据流（有后端时）

```
POST /api/search → jobId → GET /api/search/:jobId 或 WebSocket
GET /api/bottle/random
POST /api/ugc
```
