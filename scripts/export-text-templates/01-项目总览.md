# 01 · 项目总览

## 名称与定位

- **中文名**：介嘎算  
- **英文名 / 仓库代号**：ga-ge-soen、jiegasuan  
- **定位**：伪装成比价工具的本地生活情报引擎。

表面帮用户找到便宜好店；实质上通过每一次搜索、收藏、导航行为，积累本地消费需求数据，用这些数据帮商家发现经营机会、帮想开店的人找到选址依据、帮本地居民发现被埋没的好店。

这不是一个家庭自用工具，而是一个以"比价"为入口的本地商业情报网络。  
- **当前版次**：0.3.0（见 `项目根说明/VERSION`）

## 技术栈（全是可复制源码，无 Python 主程序）

| 层 | 技术 |
|----|------|
| 前端 | React 18 + Vite，Leaflet 地图，CSS 变量主题 |
| 后端 | Node.js + Express |
| 数据 | JSON 种子文件 `deals.seed.json` + 内存 UGC + 浏览器 localStorage 兜底 |
| 部署形态 | 手机浏览器打开 H5；可选本地后端 |

## 三个主 Tab（当前产品）

1. **奥扫** `AoSoPage.jsx` — 选场景/子类 → 距离/排序 → 搜索 → 卡片列表 → 详情/比价/导航  
2. **暇兜兜** `XiaDouDouPage.jsx` — 选类型 → 捞漂流瓶 → 看店家详情（环境/价格/服务）  
3. **我的** `ProfilePage.jsx` — 收藏、捞瓶历史、许愿、**上传家人发现的优惠**

## 价值维度（搜索排序）

- **比便宜**（cheap / saving）  
- **比性价比**（value，默认）  
- **比质价比**（experience / blogger）

## 场景分类（奥扫顶部）

咖啡、宠物、临期、民宿、娱乐、生活；子类见 `sceneTaxonomy.js`（如民宿：海景/田园/亲子/宠物友好/几人免几）。
