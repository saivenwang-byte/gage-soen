# 介嘎算 — 风险预警清单

> 家庭自用项目的技术、合规与产品风险。部署或对外分享前请逐项确认。

## 🔴 高优先级

### 1. 爬虫合规与账号安全

| 风险 | 说明 | 建议 |
|------|------|------|
| 违反平台 ToS | 大众点评、小红书、美团等均限制未授权抓取 | 仅家庭自用；控制频率；勿对外商用 |
| 账号封禁 | 高频 / 无 Cookie / 数据中心 IP 易触发风控 | 配置 Cookie + 住宅代理；`npm test` 验证 |
| 法律争议 | 批量抓取用户生成内容可能涉及著作权 | 不缓存全文、不二次分发；仅展示摘要与链接 |

### 2. 数据真实性

| 风险 | 说明 | 建议 |
|------|------|------|
| 回退数据冒充真实 | 无 Cookie 时 `dataMode: fallback`，UI 未标注易误导 | 前端应展示「演示数据」标签（待做） |
| 解析不准 | HTML 正则解析，页面改版即失效 | 定期 `npm test`；真实 Cookie 后人工 spot check |
| 价格过期 | 优惠有时效，爬虫非实时 | 结果页注明抓取时间；勿作为交易依据 |

### 3. 密钥与隐私

| 风险 | 说明 | 建议 |
|------|------|------|
| Cookie 泄露 | `data/cookies/*.json` 等同登录态 | 已 gitignore；勿上传网盘/截图 |
| `.env` 误提交 | 含代理、打码 API Key | 仅使用 `.env.example` 模板 |
| 定位隐私 | 前端请求 geolocation | 家庭内可接受；对外需隐私说明 |

## 🟠 中优先级

### 4. 架构与稳定性

| 风险 | 说明 | 建议 |
|------|------|------|
| 搜索任务仅内存 | 重启后端丢失 job；无 Redis 时 Bull 未启用 | 单机家庭可接受；多实例需 Redis |
| 无鉴权 API | 局域网内 `POST /api/search` 可被任意调用 | 勿暴露公网；或加简单 Token |
| 轮询 48s 上限 | 前端最多轮询 ~48s，慢爬虫可能未完成 | 已支持 WebSocket，前端可改接 WS |
| Playwright 资源 | 并发爬取吃 CPU/内存 | 家庭机单任务即可；Docker 需装 Chromium |

### 5. 产品未完成项

| 项 | 状态 |
|----|------|
| 挂机关注 → 自动推送 | 仅本地列表，无后端调度 |
| 对比 3 项 | 可选中，无对比弹窗/页 |
| 日历 / 人数筛选 | 组件存在，未接入四 Tab 奥扫 |
| 微信小程序 | `legacy-miniprogram` 与 H5 未统一 |
| 签文 UI `FortunePaper` | 暇兜兜为简易文本展示 |

## 🟡 低优先级 / 技术债

- 仓库尚未首次 commit，无版本回溯点  
- `frontend/dist` 若误提交会膨胀仓库（已在 gitignore）  
- Leaflet 依赖 unpkg CDN，离线或内网需改本地包  
- 路径含中文，部分 CI/旧工具链可能异常  

## 建议的下一步（按价值排序）

1. **Cookie + `npm test`** — 确认至少一条 `dataMode: real`  
2. **结果卡片标注 `fallback` / `real`** — 避免家人误信演示价  
3. **首次 git commit** — 锁定当前可运行基线  
4. **奥扫接入 DateCalendar + HeadcountPicker** — 复用已有组件  
5. **公网部署前** — 加 API Token、HTTPS、关闭或限制爬虫
