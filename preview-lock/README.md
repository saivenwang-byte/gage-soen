# 锁定预览（沟通用「最终界面」）

本目录存放**当前认定可对外展示**的前端构建结果，与开发主线 `frontend\` 分离。

| 文件 | 说明 |
|------|------|
| `dist\` | 执行 `scripts\lock-preview.ps1` 后生成的静态页面（与线上一致） |
| `MANIFEST.json` | 版次、锁定时间、源路径、变更摘要 |
| `RELEASE.md` | 给人看的本版说明（微信/家人沟通可直接发） |

## 怎么用

1. 开发满意后，在仓库根目录执行：`.\scripts\lock-preview.ps1`
2. 双击 **`打开介嘎算-锁定预览.bat`** → 浏览器打开 http://localhost:5174（只看锁定版，不会随改代码变）
3. 日常改代码仍用 **`打开介嘎算界面.bat`** → http://localhost:5173（开发版）

查历史版次与路径：双击 **`打开版本溯源台账.bat`**，或打开 `versions\index.html`。
